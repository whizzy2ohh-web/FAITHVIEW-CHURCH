üöÄ How to Host on GitHub Pages (5-Minute Setup)

You can host this website for free using GitHub Pages. Follow these steps:

Create a Repository:

Go to GitHub and create a new repository (e.g., faithview-website).

Optional: If you want the URL to be yourusername.github.io, name the repository exactly that.

Upload Files:

Upload the index.html file from this folder to your repository.

Activate GitHub Pages:

Go to your repository Settings.

Click on Pages in the left sidebar.

Under Build and deployment > Branch, select main (or master) and click Save.

Go Live:

Wait about 1-2 minutes. Refresh the page, and GitHub will provide your live website link (e.g., https://yourusername.github.io/faithview-website/).
